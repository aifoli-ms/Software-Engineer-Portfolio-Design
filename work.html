01BACKEND

SWIFT Transport — Bus Booking Platform

Backend APIs powering real-time trip search, booking, and digital ticketing for 19+ bus operators across Ghana.

Live ↗
PROBLEM

Bus travel in Ghana was fragmented: paper tickets, no price comparison, no seat selection. Operators had no shared digital infrastructure, and passengers had no way to compare options or book ahead reliably.

SOLUTION

Built the Route & Transit Management API from scratch: bus stop CRUD, route segment pricing, scheduling, and state management. Added UserExistsByIdGuard for secure user-scoped routes, enforced runtime env validation, and introduced CI coverage thresholds.

TECHNICAL CHALLENGES
  • Designing a flexible route segment pricing model for diverse operator needs.
  • Standardizing input validation across payment and payout endpoints.
  • Establishing CI quality gates from zero (statement 69%, branch 48%, function 46%, line 68%).
KEY FEATURES
  • Full route management API: stops, segments, pricing, schedules.
  • Driver-to-vehicle assignment for bus owner organisations.
  • Automated daily payouts (currently shipping).

02FULL-STACK / AI

Casa360 — AI Property Management

AI-powered platform for tenant management, automated rent collection, and vendor matching.

Live ↗
PROBLEM

Property owners juggled tenants, payments, and maintenance across different tools. Service providers struggled with inconsistent jobs, low visibility, and delayed payments. The system didn't work smoothly for either side.

SOLUTION

Full-stack SaaS platform with AI assistant (Cassie) for automated workflows: tenant onboarding, rent follow-ups, vendor matching, and real-time financial dashboards. Shipped 25+ bug fixes spanning transaction safety, Docker hardening, and React Query cache correctness.

TECHNICAL CHALLENGES
  • Fixed TOCTOU race conditions in payment processing (transaction safety).
  • Resolved Docker signal-forwarding and graceful shutdown for multi-replica deployments.
  • Hardened bulk import atomicity and multi-org auth context.
KEY FEATURES
  • React Query cache invalidation fixes across vendor, payment, and invoice flows.
  • Secured production by gating Swagger docs and preventing DTO field injection.
  • AI-powered tenant workflows, vendor matching, and financial reporting.

Ashesi Logistics — Operations Dashboards

Suite of 3 dashboards for resource monitoring, maintenance tracking, and campus issue reporting.

PROBLEM

The university logistics team lacked real-time visibility into utility systems. Maintenance tasks were tracked manually with no automated reminders, and students had no direct channel to report location-specific issues on campus.

SOLUTION

Built three interconnected dashboards: real-time resource monitoring for 4 water tanks and 2 treatment pipes, a preventive maintenance system managing 228 scheduled tasks with automated daily email alerts, and an issue-reporting portal with a 360° virtual campus tour for location-pinned reports.

TECHNICAL CHALLENGES
  • Integrating real-time sensor data into a responsive React dashboard.
  • Automating daily maintenance emails to responsible personnel reliably.
  • Building a 360° virtual tour with interactive issue-pinning on specific locations.
KEY FEATURES
  • Real-time utility monitoring with actionable insights.
  • 228 scheduled maintenance tasks with zero-miss email alerts.
  • 360° campus tour with student-driven issue reporting.

06CO-FOUNDER / FULL-STACK

Code Endelea — Interactive Coding Education

One platform where students watch, pause, code, and run, with an AI tutor guiding every step.

Live ↗
PROBLEM

Students learning to code constantly switch between tutorial videos, IDEs, and AI tools across multiple tabs. This breaks focus, kills momentum, and makes learning harder than it needs to be, especially for beginners at universities and bootcamps.

SOLUTION

Built an all-in-one platform where students watch guided tutorials, pause to write real code in a browser-based IDE, run it instantly, and get help from Endi, an AI tutor that guides rather than gives answers. Everything happens in a single tab with zero setup required.

TECHNICAL CHALLENGES
  • Building a browser-based code execution environment supporting Python, JavaScript, Java, C++, Rust, and Go.
  • Implementing payment gating and video locking for monetized content.
  • Auditing the video processing pipeline and benchmarking processing times for optimization.
KEY FEATURES
  • Integrated video + IDE + AI tutor in a single tab.
  • Payment and video locking for content monetization.
  • OWASP Top 10 security audit and cookie/caching hardening.
