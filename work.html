07FULL-STACK / AISolo Project

Venu — Restaurant Reservation Platform

A full-stack restaurant discovery and booking platform for Ghana with an AI concierge powered by Gemini.

Live ↗
PROBLEM

Dining out in Accra means scrolling through fragmented Instagram pages, WhatsApp groups, and word-of-mouth, then calling to check availability. Restaurant owners manage reservations through notebooks, DMs, and missed calls, losing customers to friction they can't measure.

SOLUTION

Full-stack platform connecting diners with restaurants through discovery, booking, and an AI concierge (Alfred) powered by Gemini. Alfred reasons across live database state (availability, menus, user history) and books tables directly via function calling. Owners get a dashboard for menus, reservations, and listings. Admins get a control panel for user management and moderation.

TECHNICAL CHALLENGES
  • Building an AI concierge that reasons across live DB state and writes reservations via Gemini function calling.
  • Full architectural migration from PHP/MySQL to Node.js/PostgreSQL: seven controllers, JWT auth, rate limiter, and DB abstraction rewritten for serverless.
  • Role-based access control across every endpoint: customers, owners, and admins with strict identity and ownership checks.
KEY FEATURES
  • AI concierge (Alfred) that collapses discovery-to-booking into a single conversation.
  • Restaurant owner dashboard for menus, reservations, and listing management.
  • Admin panel for user management, role assignments, and review moderation.
