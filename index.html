01BACKEND

SWIFT Transport — Bus Booking Platform

Backend APIs powering real-time trip search, booking, and digital ticketing for 19+ bus operators across Ghana.

Live ↗
PROBLEM

Bus travel in Ghana was fragmented: paper tickets, no price comparison, no seat selection. Operators had no shared digital infrastructure, and passengers had no way to compare options or book ahead reliably.

SOLUTION

Built the Route & Transit Management API from scratch: bus stop CRUD, route segment pricing, scheduling, and state management. Added UserExistsByIdGuard for secure user-scoped routes, enforced runtime env validation, and introduced CI coverage thresholds.

TECHNICAL CHALLENGES
  • Designing a flexible route segment pricing model for diverse operator needs.
  • Standardizing input validation across payment and payout endpoints.
  • Establishing CI quality gates from zero (statement 69%, branch 48%, function 46%, line 68%).
KEY FEATURES
  • Full route management API: stops, segments, pricing, schedules.
  • Driver-to-vehicle assignment for bus owner organisations.
  • Automated daily payouts (currently shipping).

06CO-FOUNDER / FULL-STACK

Code Endelea — Interactive Coding Education

One platform where students watch, pause, code, and run, with an AI tutor guiding every step.

Live ↗
PROBLEM

Students learning to code constantly switch between tutorial videos, IDEs, and AI tools across multiple tabs. This breaks focus, kills momentum, and makes learning harder than it needs to be, especially for beginners at universities and bootcamps.

SOLUTION

Built an all-in-one platform where students watch guided tutorials, pause to write real code in a browser-based IDE, run it instantly, and get help from Endi, an AI tutor that guides rather than gives answers. Everything happens in a single tab with zero setup required.

TECHNICAL CHALLENGES
  • Building a browser-based code execution environment supporting Python, JavaScript, Java, C++, Rust, and Go.
  • Implementing payment gating and video locking for monetized content.
  • Auditing the video processing pipeline and benchmarking processing times for optimization.
KEY FEATURES
  • Integrated video + IDE + AI tutor in a single tab.
  • Payment and video locking for content monetization.
  • OWASP Top 10 security audit and cookie/caching hardening.
